# RTF decomposer

Public tool for bulk extracting various embedded object types from a given RTF file or a whole folder repository.

## Presentation :

I've decided to release this tool to the public in order to assist fellow Malware analysts & Antivirus engineers at their daily tasks while dealing whith RTF objects. As you perfectly know : manual extraction of RTF embedded data is such a painful process. As for the other public alternatives like <b>rtfobj</b>
